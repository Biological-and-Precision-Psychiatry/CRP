# Get First or Last Element

Get First or Last Element

## Usage

``` r
first(x)

last(x)
```

## Arguments

- x:

  a vector

## Value

the first or last element

## Author

Rune Haubo B Christensen

## Examples

``` r
x <- rnorm(5)
x
#> [1] -0.005571287  0.621552721  1.148411606 -1.821817661 -0.247325302
first(x)
#> [1] -0.005571287
last(x)
#> [1] -0.2473253
```
