# Package index

## Functions that format

- [`do_format()`](https://biological-and-precision-psychiatry.github.io/CRP/reference/do_format.md)

- [`format_estci()`](https://biological-and-precision-psychiatry.github.io/CRP/reference/format_estci.md)
  : Format estimate with confidence interval
- [`format_pval2()`](https://biological-and-precision-psychiatry.github.io/CRP/reference/format_pval2.md)
  : Format p-values consistently
- [`comma2numeric()`](https://biological-and-precision-psychiatry.github.io/CRP/reference/comma2numeric.md)

- [`show_na()`](https://biological-and-precision-psychiatry.github.io/CRP/reference/show_na.md)
